== Description ==

The plugin allows Woocommerce users to easily see if there are products which do not have manual prices set for additional currencies created through WPML.

For each product or variation which is listed, click on the product title to edit this product and set manual prices for additional currencies.

The plugin helps prevent accidentally under-charging for products by using an outdated or incorrect exchange rate, by highlighting and encouraging users to set manual prices for all currencies.

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip the woocommerce-prices-in-other-currencies.zip file
2. Upload the `woocommerce-prices-in-other-crrencies` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
